### 1. الحمى (Fever)

**المؤشرات الحيوية الحرجة:**
- درجة الحرارة ≥ 40.0°م (104°ف) عند أي عمر
- درجة الحرارة ≥ 38.0°م في رضيع < 3 أشهر
- نبضات القلب > 150 نبضة/دقيقة (بالغين)
- ضغط الدم الانقباضي < 90 ملم زئبق
- تشبع الأكسجين (SpO2) < 92%

- **الحالة الحرجة (Critical):**
  - حمى ≥ 40.0°م مصحوبة بتشنجات، أو غيبوبة، أو صعوبة في التنفس، أو طفح جلدي لا يختفي عند الضغط (Purpura)، أو تقيؤ مستمر، أو صداع شديد مع تقيؤ، أو رقبة متصلبة، أو هبوط ضغط الدم (الانقباضي < 90 ملم زئبق)، أو حمى في رضيع < 3 أشهر ≥ 38.0°م.
  - **النصيحة الطبية:** التوجه فوراً إلى قسم الطوارئ. لا تنتظر.

- **الحالة عالية الخطورة (High):**
  - حمى 39.0–39.9°م مصحوبة بصداع شديد، أو ألم حلقي شديد مع صعوبة البلع، أو ألم صدري، أو ضيق تنفس خفيف، أو تقيؤ متكرر، أو إسهال مدمّي، أو حمى في طفل 3–6 أشهر ≥ 38.5°م، أو حمى مستمرة > 5 أيام.
  - **النصيحة الطبية:** زيارة عيادة طبية عاجلة أو طوارئ غير حرجة اليوم. يُفضّل خلال 4–6 ساعات.

- **الحالة المتوسطة (Medium):**
  - حمى 38.0–38.9°م مصحوبة بسعال جاف، أو ألم حلقي خفيف، أو آلام جسدية عامة، أو إسهال بسيط، أو استمرار الحمى 2–3 أيام دون تحسن.
  - **النصيحة الطبية:** حجز موعد مع طبيب عام أو عيادة خلال 24–48 ساعة. مراقبة الحرارة والأعراض.

- **الحالة غير الطارئة (Low):**
  - حمى خفيفة < 38.0°م منفردة، دون أعراض مصاحبة خطيرة، في شخص بالغ بصحة جيدة.
  - **النصيحة الطبية:** الراحة في المنزل، شرب السوائل، تناول مسكنات الحمى (باراسيتامول 500–1000 مجم كل 6–8 ساعات). مراجعة الطبيب إذا استمرت > 3 أيام.

### 2. ألم الصدر (Chest Pain)

**المؤشرات الحيوية الحرجة:**
- ضغط الدم الانقباضي < 90 أو > 180 ملم زئبق
- نبضات القلب < 40 أو > 150 نبضة/دقيقة
- تشبع الأكسجين (SpO2) < 90%
- معدل التنفس > 30 أنفاس/دقيقة

- **الحالة الحرجة (Critical):**
  - ألم صدري حاد مثل "السحق" (Crushing) يمتد إلى الذراع الأيسر أو الفك أو الظهر، أو مصحوب بعرق بارد، أو غثيان/تقيؤ، أو إغماء، أو ضيق تنفس شديد، أو نبضات قلب غير منتظمة سريعة، أو ألم صدري بعد صدمة صدرية (حادث سير)، أو ألم صدري مع كحة دموية.
  - **النصيحة الطبية:** استدعاء الإسعاف (911/999) فوراً. لا تتحرك. مضغ أسبرين 325 مجم إذا لم يكن هناك حساسية.

- **الحالة عالية الخطورة (High):**
  - ألم صدري متوسط الشدة يزداد مع الجهد، أو مصحوب بضيق تنفس خفيف، أو خفقان قلب، أو ألم حاد جانبي الصدر مع صعوبة التنفس العميق (احتمال انصمام رئوي)، أو ألم صدري في مريض معروف بأمراض القلب.
  - **النصيحة الطبية:** زيارة طوارئ أو عيادة قلبية عاجلة اليوم. رسم قلب (ECG) ضروري.

- **الحالة المتوسطة (Medium):**
  - ألم صدري خفيف يتغير بالوضعية، أو يرتبط بالحركة/الضغط على الصدر (عصلي)، أو مصحوب بحموضة/ارتجاع، أو استمرار > 24 ساعة دون تفاقم.
  - **النصيحة الطبية:** حجز موعد مع طبيب باطنية أو قلبية خلال 24–48 ساعة. تجنب الجهد البدني.

- **الحالة غير الطارئة (Low):**
  - ألم صدري عرضي خفيف، مرتبط بالتوتر أو العضلات، يتحسن بالراحة، لا يوجد أعراض مصاحبة.
  - **النصيحة الطبية:** الراحة، تطبيق كمادات دافئة، مسكنات مثل Ibuprofen 400 مجم. مراجعة الطبيب إذا تكرر.

### 3. الصداع (Headache)

**المؤشرات الحيوية الحرجة:**
- درجة الحرارة ≥ 40.0°م مع صداع
- ضغط الدم الانقباضي > 180 ملم زئبق مع صداع
- نبضات القلب > 120 نبضة/دقيقة مع صداع شديد
- درجة وعي مُنخفضة (GCS < 15)

- **الحالة الحرجة (Critical):**
  - "أسوأ صداع في الحياة" فجائي الحدوث (Thunderclap)، أو صداع مصحوب بفقدان الوعي، أو تشنجات، أو ضعف/خدر في أحد جانبي الجسم، أو صعوبة في الكلام، أو اختلال الرؤية المفاجئ، أو تقيؤ مستمر، أو رقبة متصلبة مع حمى، أو صداع بعد إصابة رأسية (حادث/سقوط)، أو صداع في حامل ≥ 20 أسبوع مع ارتفاع ضغط الدم.
  - **النصيحة الطبية:** التوجه فوراً إلى الطوارئ. تصوير مقطعي (CT) للدماغ ضروري.

- **الحالة عالية الخطورة (High):**
  - صداع شديد جديد النمط (لم يحدث من قبل)، أو صداع مستمر > 72 ساعة دون تحسن، أو صداع مصحوب بارتفاع ضغط الدم (> 160/100)، أو صداع في مريض سرطان أو نقص مناعة، أو صداع مع تورم في العين/احمرار.
  - **النصيحة الطبية:** زيارة طوارئ أو عيادة عصبية عاجلة اليوم. تقييم عصبي شامل مطلوب.

- **الحالة المتوسطة (Medium):**
  - صداع متكرر يؤثر على النشاط اليومي، أو صداع مصحوب بغثيان خفيف، أو حساسية للضوء/الصوت (احتمال شقيقة)، أو صداع يستمر > 24 ساعة.
  - **النصيحة الطبية:** حجز موعد مع طبيب عام أو عصبية خلال 24–48 ساعة. تدوين تكرار الصداع.

- **الحالة غير الطارئة (Low):**
  - صداع خفيف عرضي، مرتبط بالإجهاد، أو قلة النوم، أو الجفاف، يتحسن بالمسكنات.
  - **النصيحة الطبية:** شرب الماء، الراحة في غرفة مظلمة، تناول Paracetamol 500–1000 مجم أو Ibuprofen 400 مجم. مراجعة الطبيب إذا تكرر > 4 مرات/شهر.

### 4. ألم البطن (Abdominal Pain)

**المؤشرات الحيوية الحرجة:**
- ضغط الدم الانقباضي < 90 ملم زئبق
- نبضات القلب > 120 نبضة/دقيقة
- درجة الحرارة ≥ 39.5°م
- معدل التنفس > 30 أنفاس/دقيقة
- بطن صلب/متصلب (Rigid abdomen)

- **الحالة الحرجة (Critical):**
  - ألم بطن شديد مفاجئ مع بطن صلب (صلابة دفاعية)، أو تقيؤ دموي، أو براز أسود/دمي، أو ألم بطن مصحوب بفقدان الوعي، أو انتفاخ شديد مع عدم خروج غازات/براز (احتمال انسداد)، أو ألم بطن شديد في حامل مع نزيف، أو ألم في الجزء السفلي الأيمن مع حمى ≥ 39°م (احتمال انفجار زائدة دودية).
  - **النصيحة الطبية:** التوجه فوراً إلى الطوارئ. لا تأكل أو تشرب. جراحة عاجلة محتملة.

- **الحالة عالية الخطورة (High):**
  - ألم بطن متوسط إلى شديد يستمر > 6 ساعات، أو مصحوب بحمى 38.5–39.4°م، أو تقيؤ متكرر يمنع شرب السوائل، أو ألم في الجزء السفلي الأيمن/الأيسر مع حمى، أو ألم بطن في مريض مسن (> 65 سنة)، أو ألم فوق السرة مع غثيان (احتمال التهاب المرارة/البنكرياس).
  - **النصيحة الطبية:** زيارة طوارئ أو عيادة جراحية عاجلة اليوم. صيام حتى التقييم.

- **الحالة المتوسطة (Medium):**
  - ألم بطن خفيف إلى متوسط يستمر > 24 ساعة، أو مصحوب بإسهال/إمساك، أو غازات مؤلمة، أو ألم متكرر بعد الأكل (احتمال قرحة/التهاب).
  - **النصيحة الطبية:** حجز موعد مع طبيب باطنية أو جراحة عامة خلال 24–48 ساعة. تجنب الأطعمة الدسمة.

- **الحالة غير الطارئة (Low):**
  - ألم بطن خفيف عرضي، مرتبط بالغازات، أو الإمساك المؤقت، أو الحيض، يتحسن بالمسكنات.
  - **النصيحة الطبية:** شرب السوائل الدافئة، تناول Paracetamol، تطبيق كمادات دافئة. مراجعة الطبيب إذا استمر > 48 ساعة.

### 5. ضيق التنفس (Shortness of Breath / Dyspnea)

**المؤشرات الحيوية الحرجة:**
- تشبع الأكسجين (SpO2) < 88% على الهواء الطبيعي
- معدل التنفس > 30 أنفاس/دقيقة
- نبضات القلب > 150 نبضة/دقيقة
- استخدام عضلات مساعدة للتنفس (الترقوة، الفجوة فوق القص)
- عدم القدرة على الكلام بجمل كاملة

- **الحالة الحرجة (Critical):**
  - ضيق تنفس شديد يمنع الكلام، أو SpO2 < 88%، أو شفاه/أظافر زرقاء (Cyanosis)، أو ألم صدري حاد مع ضيق تنفس، أو كحة دموية، أو فقدان الوعي، أو صوت تنفسي عالٍ (Stridor)، أو تورم في الوجه/اللسان/الحلق (احتمال وذمة وعائية).
  - **النصيحة الطبية:** استدعاء الإسعاف فوراً. الجلوس في وضع مستقيم. أكسجين إذا متوفر.

- **الحالة عالية الخطورة (High):**
  - ضيق تنفس متوسط يزداد مع الجهد أو الاستلقاء، أو SpO2 88–92%، أو ضيق تنفس مصحوب بألم صدري، أو خفقان، أو تورم في الساقين (احتمال خثار/فشل قلبي)، أو ضيق تنفس في مريض ربو/انسداد رئوي مزمن مع عدم استجابة للبخاخ.
  - **النصيحة الطبية:** زيارة طوارئ أو عيادة صدرية عاجلة اليوم. تقييم أشعة سينية ووظائف رئة.

- **الحالة المتوسطة (Medium):**
  - ضيق تنفس خفيف يظهر مع الجهد فقط، أو سعال مصحوب ببلغم، أو ضيق تنفس مصحوب بأنفلونزا، أو ضيق تنفس متكرر في مريض ربو مع استجابة جزئية للبخاخ.
  - **النصيحة الطبية:** حجز موعد مع طبيب باطنية أو صدرية خلال 24–48 ساعة. مراقبة SpO2 إذا متوفر.

- **الحالة غير الطارئة (Low):**
  - ضيق تنفس خفيف عرضي، مرتبط بالقلق، أو التوتر، أو التعب، أو البدانة، بدون أعراض مصاحبة خطيرة.
  - **النصيحة الطبية:** تمارين تنفس عميقة، تجنب المجهود المفاجئ، شرب السوائل. مراجعة الطبيب إذا تكرر أو استمر.

### 6. الدوخة والإغماء (Dizziness & Syncope)

**المؤشرات الحيوية الحرجة:**
- ضغط الدم الانقباضي < 80 ملم زئبق
- نبضات القلب < 40 أو > 150 نبضة/دقيقة
- درجة الحرارة ≥ 40.0°م
- GCS (مقياس غلاسكو) < 15

- **الحالة الحرجة (Critical):**
  - إغماء كامل (Syncope) مصحوب بإصابة رأسية، أو فقدان الوعي > 5 دقائق، أو إغماء مصحوب بتشنجات، أو ضعف/خدر في جانب واحد، أو صعوبة كلام، أو دوخة مع ألم صدري حاد، أو دوخة مع كحة دموية، أو دوخة شديدة مع رؤية مزدوجة وعدم اتزان (احتمال سكتة دماغية).
  - **النصيحة الطبية:** استدعاء الإسعاف فوراً. لا تترك المريض وحده. تقييم عصبي ودماغي طارئ.

- **الحالة عالية الخطورة (High):**
  - إغماء قصير (< 1 دقيقة) بدون إصابة، أو دوخة شديدة مستمرة > 1 ساعة، أو دوخة مصحوبة بخفقان قلب، أو دوخة مع صداع شديد جديد، أو دوخة في مريض قلبي/سكري/مسن (> 70 سنة)، أو دوخة مع تقيؤ مستمر.
  - **النصيحة الطبية:** زيارة طوارئ أو عيادة عصبية/قلبية عاجلة اليوم. تقييم شامل للقلب والأذن.

- **الحالة المتوسطة (Medium):**
  - دوخة خفيفة متكررة، أو دوخة عند الوقوف السريع (Orthostatic)، أو دوخة مصحوبة بصداع خفيف، أو دوخة في مريض فقر دم معروف.
  - **النصيحة الطبية:** حجز موعد مع طبيب عام أو عصبية/أنف وأذن خلال 24–48 ساعة. قياس ضغط الدم واقفاً وجالساً.

- **الحالة غير الطارئة (Low):**
  - دوخة خفيفة عرضية، مرتبطة بالإرهاق، أو الجوع، أو قلة النوم، أو الجفاف، تختفي بالراحة.
  - **النصيحة الطبية:** شرب السوائل، تناول وجبة خفيفة، النوم الكافي. مراجعة الطبيب إذا تكررت > 3 مرات/أسبوع.

### 7. السعال (Cough)

**المؤشرات الحيوية الحرجة:**
- SpO2 < 88%
- معدل التنفس > 30 أنفاس/دقيقة
- نبضات القلب > 150 نبضة/دقيقة
- درجة الحرارة ≥ 40.0°م

- **الحالة الحرجة (Critical):**
  - سعال مصحوب بدم (Hemoptysis) كميات كبيرة، أو سعال مع ضيق تنفس شديد وSpO2 < 88%، أو سعال مع ألم صدري حاد وضيق تنفس (احتمال انصمام رئوي)، أو سعال مع تورم في الوجه/الحلق/صعوبة البلع (احتمال حساسية/وذمة وعائية).
  - **النصيحة الطبية:** استدعاء الإسعاف فوراً. الجلوس مستقيماً. لا تأخذ أي سوائل.

- **الحالة عالية الخطورة (High):**
  - سعال مستمر > 3 أسابيع، أو سعال مصحوب ببلغم أصفر/أخضر مع حمى ≥ 38.5°م، أو سعال مع ضيق تنفس متوسط، أو سعال مع فقدان وزن غير مبرر، أو سعال دموي بكميات قليلة، أو سعال في مريض سرطان/نقص مناعة.
  - **النصيحة الطبية:** زيارة طوارئ أو عيادة صدرية عاجلة اليوم. أشعة سينية للصدر مطلوبة.

- **الحالة المتوسطة (Medium):**
  - سعال مصحوب ببلغم شفاف/أبيض، أو سعال مع حمى خفيفة 37.5–38.4°م، أو سعال ليلي متكرر، أو سعال مع حرقة/ارتجاع.
  - **النصيحة الطبية:** حجز موعد مع طبيب باطنية أو صدرية خلال 24–48 ساعة. تجنب التدخين والمهيجات.

- **الحالة غير الطارئة (Low):**
  - سعال خفيف عرضي، بدون بلغم أو حمى، مرتبط بالحساسية الموسمية، أو التهيج بالغبار، أو نزلة برد خفيفة.
  - **النصيحة الطبية:** شرب السوائل الدافئة، عسل طبيعي، تجنب المهيجات. مراجعة الطبيب إذا استمر > 2 أسابيع.

### 8. التهاب الحلق (Sore Throat / Pharyngitis)

**المؤشرات الحيوية الحرجة:**
- درجة الحرارة ≥ 40.0°م
- نبضات القلب > 150 نبضة/دقيقة
- SpO2 < 92% (احتمال انسداد مجاري هوائية)
- صعوبة التنفس/البلع الشديدة

- **الحالة الحرجة (Critical):**
  - التهاب حلق شديد مع صعوبة تنفس شديدة، أو تورم في الحلق/اللسان يمنع البلع، أو صوت "خرخرة" (Stridor)، أو التهاب حلق مع رقبة متصلبة وحمى ≥ 39.5°م (احتمال خناق/إبiglottitis)، أو فقدان الوعي.
  - **النصيحة الطبية:** استدعاء الإسعاف فوراً. لا تحاول فحص الحلق بالمنزل. قد يحتاج تدخل جراحي طارئ.

- **الحالة عالية الخطورة (High):**
  - التهاب حلق شديد مع صعوبة بلع اللعاب، أو تورم في الغدد اللمفاوية كبير جداً، أو حمى ≥ 39.0°م مع تقيؤ، أو طفح جلدي أحمر خشن (احتمال حمى القرمزية)، أو التهاب حلق في مريض نقص مناعة/سرطان، أو ألم حلق مع تنفس صفير (Wheezing).
  - **النصيحة الطبية:** زيارة طوارئ أو عيادة أنف وأذن عاجلة اليوم. مسحة حلق وفحص بكتيري مطلوب.

- **الحالة المتوسطة (Medium):**
  - التهاب حلق مع حمى 38.0–38.9°م، أو بلغم أصفر، أو ألم أذن مصاحب، أو استمرار > 3 أيام دون تحسن، أو صعوبة بلع خفيفة.
  - **النصيحة الطبية:** حجز موعد مع طبيب عام أو أنف وأذن خلال 24–48 ساعة. مسحة حلق للاستبعاد البكتيري.

- **الحالة غير الطارئة (Low):**
  - التهاب حلق خفيف، بدون حمى، أو مع حمى < 38.0°م، يتحسن بالغرغرة والسوائل الدافئة.
  - **النصيحة الطبية:** غرغرة بالماء المالح الدافئ، شرب السوائل، راحة الصوت، مسكنات (Paracetamol). مراجعة الطبيب إذا استمر > 5 أيام.

### 9. ألم الظهر (Back Pain)

**المؤشرات الحيوية الحرجة:**
- ضغط الدم الانقباضي < 90 ملم زئبق
- درجة الحرارة ≥ 39.5°م مع ألم ظهر
- فقدان السيطرة على المثانة/الأمعاء
- ضعف/شلل في الساقين

- **الحالة الحرجة (Critical):**
  - ألم ظهر شديد مع فقدان السيطرة على البول/البراز (احتمال متلازمة ذنب الفرس)، أو ألم ظهر مع ضعف/شلل في الساقين، أو ألم ظهر بعد حادث/سقوط مع فقدان الإحساس، أو ألم ظهر مع حمى ≥ 39.5°م وتشنجات، أو ألم ظهر مصحوب بألم صدري/ضيق تنفس.
  - **النصيحة الطبية:** استدعاء الإسعاف فوراً. لا تحرك المريض. جراحة عصبية طارئة محتملة.

- **الحالة عالية الخطورة (High):**
  - ألم ظهر شديد يمتد إلى الساق مع خدر/ضعف (عرق النسا)، أو ألم ظهر مستمر > 2 أسابيع دون تحسن، أو ألم ظهر في مريض سرطان/نقص مناعة، أو ألم ظهر مع حمى 38.5–39.4°م (احتمال عدوى عمود فقري)، أو ألم ظهر مع فقدان وزن غير مبرر.
  - **النصيحة الطبية:** زيارة طوارئ أو عيادة عصبية/عظام عاجلة اليوم. تصوير رنين مغناطيسي (MRI) قد يكون مطلوباً.

- **الحالة المتوسطة (Medium):**
  - ألم ظهر متوسط يستمر > 1 أسبوع، أو مصحوب بتصلب صباحي، أو ألم يزداد مع الحركة، أو ألم ظهر متكرر > 3 مرات/شهر.
  - **النصيحة الطبية:** حجز موعد مع طبيب عظام أو عصبية أو علاج طبيعي خلال 24–48 ساعة. تجنب رفع الأثقال.

- **الحالة غير الطارئة (Low):**
  - ألم ظهر خفيف عرضي، مرتبط بالحركة الخاطئة، أو الجلوس الطويل، أو التوتر العضلي، يتحسن بالراحة.
  - **النصيحة الطبية:** الراحة، تطبيق كمادات دافئة/باردة، تمارين stretching خفيفة، مسكنات (Ibuprofen 400 مجم). مراجعة الطبيب إذا استمر > 1 أسبوع.

### 10. الغثيان والقيء (Nausea & Vomiting)

**المؤشرات الحيوية الحرجة:**
- ضغط الدم الانقباضي < 90 ملم زئبق
- نبضات القلب > 150 نبضة/دقيقة
- درجة الحرارة ≥ 40.0°م
- جفاف شديد (تبول < 3 مرات/يوم، لسان جاف)

- **الحالة الحرجة (Critical):**
  - تقيؤ دموي (أحمر/قهوة)، أو تقيؤ مستمر يمنع شرب أي سوائل > 12 ساعة، أو تقيء مصحوب بألم بطن شديد وصلابة، أو تقيء مصحوب بفقدان الوعي، أو تقيء مع صداع شديد ورقبة متصلبة، أو تقيء في رضيع < 6 أشهر مع انتفاخ البطن (احتمال انسداد).
  - **النصيحة الطبية:** استدعاء الإسعاف فوراً. خطر جفاف شديد ومضاعفات حادة.

- **الحالة عالية الخطورة (High):**
  - تقيء متكرر > 6 مرات/يوم لمدة > 24 ساعة، أو تقيء مصحوب بألم بطن متوسط، أو تقيء مع حمى ≥ 38.5°م، أو تقيء مع إسهال مدمّي، أو تقيء في حامل > 20 أسبوع (احتمال Hyperemesis Gravidarum)، أو تقيء في مريض سكري مع أعراض حمضية (Ketoacidosis).
  - **النصيحة الطبية:** زيارة طوارئ أو عيادة عاجلة اليوم. قد يحتاج سوائل وريدية.

- **الحالة المتوسطة (Medium):**
  - غثيان وتقيء 2–5 مرات/يوم لمدة 1–2 أيام، أو مصحوب بإسهال خفيف، أو غثيان بعد تناول طعام مشبوه، أو غثيان مصحوب بصداع خفيف.
  - **النصيحة الطبية:** حجز موعد مع طبيب عام أو باطنية خلال 24–48 ساعة. شرب السوائل ببطء.

- **الحالة غير الطارئة (Low):**
  - غثيان خفيف عرضي، بدون تقيء، أو تقيء 1–2 مرات، مرتبط بالتوتر، أو دوار الحركة، أو تناول طعام غريب.
  - **النصيحة الطبية:** شرب سوائل واضحة (ماء، شاي أخضر)، تناول وجبات خفيفة (توست، أرز)، راحة. مراجعة الطبيب إذا استمر > 24 ساعة.

### 11. الإسهال (Diarrhea)

**المؤشرات الحيوية الحرجة:**
- ضغط الدم الانقباضي < 90 ملم زئبق
- نبضات القلب > 150 نبضة/دقيقة
- درجة الحرارة ≥ 40.0°م
- علامات جفاف شديد: عدم تبول > 8 ساعات، لسان جاف، غيبوبة

- **الحالة الحرجة (Critical):**
  - إسهال دموي (أحمر/أسود) مع ألم بطن شديد، أو إسهال مائي > 10 مرات/يوم مع جفاف شديد، أو إسهال مع حمى ≥ 40.0°م وتشنجات، أو إسهال في رضيع < 6 أشهر مع جفاف، أو إسهال مع فقدان الوعي.
  - **النصيحة الطبية:** استدعاء الإسعاف فوراً. خطر جفاف شديد وتسمم دموي (Sepsis).

- **الحالة عالية الخطورة (High):**
  - إسهال دموي بكميات قليلة، أو إسهال مائي > 6 مرات/يوم لمدة > 24 ساعة، أو إسهال مع حمى 38.5–39.9°م، أو إسهال في مريض مسن (> 65 سنة) أو نقص مناعة، أو إسهال بعد سفر إلى بلدان وبائية.
  - **النصيحة الطبية:** زيارة طوارئ أو عيادة عاجلة اليوم. فحص براز وزراعة مطلوبة.

- **الحالة المتوسطة (Medium):**
  - إسهال 3–5 مرات/يوم لمدة 2–3 أيام، أو مصحوب بغثيان خفيف، أو إسهال مع مغص بسيط، أو إسهال بعد تناول مضاد حيوي (احتمال C. difficile).
  - **النصيحة الطبية:** حجز موعد مع طبيب باطنية أو عيادة جهاز هضمي خلال 24–48 ساعة. شرب محاليل إعادة الترطيب (ORS).

- **الحالة غير الطارئة (Low):**
  - إسهال خفيف 1–2 مرات/يوم، بدون دم أو حمى، مرتبط بتغيير في النظام الغذائي، أو إجهاد، أو حساسية طعام.
  - **النصيحة الطبية:** شرب السوائل، تجنب الألبان والدهون، تناول BRAT diet (موز، أرز، تفاح، توست). مراجعة الطبيب إذا استمر > 3 أيام.

### 12. الطفح الجلدي (Rash / Skin Eruption)

**المؤشرات الحيوية الحرجة:**
- درجة الحرارة ≥ 40.0°م مع طفح
- ضغط الدم الانقباضي < 90 ملم زئبق
- نبضات القلب > 150 نبضة/دقيقة
- SpO2 < 90% (احتمال وذمة وعائية)

- **الحالة الحرجة (Critical):**
  - طفح منتشر سريع الانتشار مع تورم في الوجه/الحلق/صعوبة التنفس (احتمال صدمة حساسية/Anaphylaxis)، أو طفح أرجواني لا يختفي بالضغط (Purpura) مع حمى (احتمال مرض مننجococcal)، أو طفح مع تقرحات فموية شديدة وعينين حمراوين (احتمال SJS/TEN)، أو طفح مع فقاعات شديدة وآلام، أو طفح في رضيع < 3 أشهر مع حمى ≥ 38.0°م.
  - **النصيحة الطبية:** استدعاء الإسعاف فوراً. صدمة حساسية أو عدوى مننجococcal تهدد الحياة.

- **الحالة عالية الخطورة (High):**
  - طفح منتشر مع حمى 38.5–39.9°م، أو طفح مؤلم/حارق، أو طفح مع تورم في الوجه/الشفتين (Angioedema) بدون صعوبة تنفس، أو طفح في مريض سرطان/نقص مناعة، أو طفح بعد تناول دواء جديد (احتمال حساسية دوائية).
  - **النصيحة الطبية:** زيارة طوارئ أو عيادة جلدية عاجلة اليوم. توقف أي دواء جديد فوراً.

- **الحالة المتوسطة (Medium):**
  - طفح خفشي (Hives) متكرر، أو طفح مع حكة شديدة، أو طفح مع حمى خفيفة < 38.5°م، أو طفح في منطقة محددة مع ألم/حرقة (احتمال هربس زوستر)، أو طفح يستمر > 1 أسبوع.
  - **النصيحة الطبية:** حجز موعد مع طبيب جلدية أو طبيب عام خلال 24–48 ساعة. تجنب الحك.

- **الحالة غير الطارئة (Low):**
  - طفح خفيف محدود، بدون حمى أو حكة شديدة، مرتبط بالحساسية الموسمية، أو لمس نبات، أو تغيير مستحضرات.
  - **النصيحة الطبية:** كريم هيدروكورتيزون 1%، مضادات الهيستامين (Cetirizine 10 مجم)، تجنب المهيجات. مراجعة الطبيب إذا انتشر.

### 13. ألم المفاصل (Joint Pain / Arthralgia)

**المؤشرات الحيوية الحرجة:**
- درجة الحرارة ≥ 40.0°م مع احمرار مفصل
- ضغط الدم الانقباضي < 90 ملم زئبق
- نبضات القلب > 150 نبضة/دقيقة
- مفصل شديد الاحمرار والتورم مع عدم القدرة على الحركة

- **الحالة الحرجة (Critical):**
  - ألم مفصل شديد مع احمرار وتورم شديد وحمى ≥ 39.5°م (احتمال التهاب مفصل Septic)، أو ألم مفصل بعد إصابة مع تشوه واضح (كسر/خلع)، أو ألم مفصل مع فقدان الوعي، أو ألم مفصل مع طفح أرجواني (احتمال مرض مننجococcal).
  - **النصيحة الطبية:** استدعاء الإسعاف فوراً. التهاب المفصل Septic يحتاج تدخل جراحي طارئ.

- **الحالة عالية الخطورة (High):**
  - ألم مفصل شديد مع احمرار وتورم مع حمى 38.5–39.4°م، أو ألم مفصل مفاجئ شديد في إصبع القدم الكبير (احتمال النقرس)، أو ألم مفصل مع تقيؤ وحمى (احتمال حمى روماتيزمية)، أو ألم مفصل في مريض بديل مفصل/زرع عضو.
  - **النصيحة الطبية:** زيارة طوارئ أو عيادة روماتيزمية/عظام عاجلة اليوم. سحب سائل مفصلي قد يكون مطلوباً.

- **الحالة المتوسطة (Medium):**
  - ألم مفصل متوسط يستمر > 1 أسبوع، أو تورم خفيف مع صباحية (تصلب صباحي > 30 دقيقة)، أو ألم مفاصل متعددة، أو ألم مفصل مع طفح جلدي.
  - **النصيحة الطبية:** حجز موعد مع طبيب روماتيزم أو عظام خلال 24–48 ساعة. تجنب الحمل على المفصل.

- **الحالة غير الطارئة (Low):**
  - ألم مفصل خفيف عرضي، مرتبط بالجهد الزائد، أو التقدم في السن، أو الطقس البارد، يتحسن بالمسكنات.
  - **النصيحة الطبية:** تطبيق كمادات دافئة/باردة، مسكنات (Ibuprofen 400 مجم)، تمارين خفيفة. مراجعة الطبيب إذا استمر > 2 أسابيع.

---

### 14. الأعراض البولية (Urinary Symptoms)

**المؤشرات الحيوية الحرجة:**
- درجة الحرارة ≥ 40.0°م مع أعراض بولية
- ضغط الدم الانقباضي < 90 ملم زئبق
- نبضات القلب > 150 نبضة/دقيقة
- عدم خروج البول > 12 ساعة مع ألم بطن

- **الحالة الحرجة (Critical):**
  - عدم القدرة على التبول مع ألم بطن شديد (احتمال انسداد/احتباس بولي)، أو بول دموي مع جلطات، أو ألم جانبي ظهر شديد مع حمى ≥ 39.5°م وتقيؤ (احتمال عدوى كلى Septic)، أو عدم تبول > 12 ساعة مع انتفاخ بطن، أو ألم بطن شديد في حامل مع دم في البول.
  - **النصيحة الطبية:** استدعاء الإسعاف فوراً. قسطرة بولية طارئة أو جراحة قد تكون مطلوبة.

- **الحالة عالية الخطورة (High):**
  - حرقة بولية شديدة مع دم في البول، أو تكرار بولي شديد مع حمى 38.5–39.4°م، أو ألم جانبي ظهر مع حمى (احتمال التهاب حوض كلى/Pyelonephritis)، أو تكرار بولي في مريض سكري/مسن/حامل، أو بول كري (احتمال جفاف شديد/فشل كلوي).
  - **النصيحة الطبية:** زيارة طوارئ أو عيادة باطنية/نسائية عاجلة اليوم. تحليل بول وزراعة مطلوبة.

- **الحالة المتوسطة (Medium):**
  - حرقة بولية خفيفة مع تكرار بولي، أو بول عكر مع رائحة كريهة، أو ألم في أسفل البطن (احتمال التهاب مثانة/Cystitis)، أو تسرب بولي خفيف.
  - **النصيحة الطبية:** حجز موعد مع طبيب باطنية أو نسائية/تناسلية خلال 24–48 ساعة. شرب كميات كبيرة من الماء.

- **الحالة غير الطارئة (Low):**
  - تكرار بولي خفيف، بدون حرقة أو دم، مرتبط بشرب سوائل كثيرة، أو قلق، أو برد.
  - **النصيحة الطبية:** شرب الماء بانتظام، تجنب الكافيين والكحول. مراجعة الطبيب إذا استمر > 3 أيام أو تكرر.

---

### 15. النزيف (Bleeding / Hemorrhage)

**المؤشرات الحيوية الحرجة:**
- ضغط الدم الانقباضي < 90 ملم زئبق
- نبضات القلب > 150 نبضة/دقيقة
- معدل التنفس > 30 أنفاس/دقيقة
- SpO2 < 90%
- فقدان وعي

- **الحالة الحرجة (Critical):**
  - نزيف حاد من أي مكان مع ضغط انقباضي < 90، أو نبض > 150، أو فقدان وعي، أو نزيف من الجرح لا يتوقف بالضغط المباشر > 10 دقائق، أو تقيؤ دموي كميات كبيرة، أو براز أسود/دمي مع ألم بطن، أو نزيف مهبلي شديد في حامل (احتمال إجهاض/انفصال المشيمة)، أو نزيف من الرأس/الوجه بعد إصابة.
  - **النصيحة الطبية:** استدعاء الإسعاف فوراً. الضغط المباشر على الجرح. رفع الساقين إذا أمكن. لا تعطي سوائل فموية إذا فاقد الوعي.

- **الحالة عالية الخطورة (High):**
  - نزيف معتدل من الجرح يتوقف بالضغط لكن يحتاج تضميد، أو نزيف أنفي شديد لا يتوقف > 20 دقيقة، أو نزيف لثوي شديد، أو نزيف مهبلي غير حامل مع كميات متوسطة، أو براز أسود بدون ألم (Melena)، أو بول دموي مع جلطات.
  - **النصيحة الطبية:** زيارة طوارئ أو عيادة عاجلة اليوم. قد يحتاج تضميد/كي أو نقل دم.

- **الحالة المتوسطة (Medium):**
  - نزيف خفيف من الجرح يتوقف بالضغط، أو نزيف أنفي خفيف يتوقف < 20 دقيقة، أو نزيف لثوي خفيف، أو كدمات متكررة غير مبررة، أو نزيف مهبلي خفيف غير منتظم.
  - **النصيحة الطبية:** حجز موعد مع طبيب عام أو جراحة/نسائية خلال 24–48 ساعة. فحص تخثر دم قد يكون مطلوباً.

- **الحالة غير الطارئة (Low):**
  - نزيف سطحي خفيف (خدش، كشط)، أو نزيف أنفي عرضي خفيف يتوقف بسرعة، أو نزيف لثوي عرضي أثناء تنظيف الأسنان.
  - **النصيحة الطبية:** غسل الجرح بالماء والصابون، تضميد نظيف، ضغط مباشر. مراجعة الطبيب إذا تكرر أو لم يلتئم.

## ملاحظات عامة على المؤشرات الحيوية (Vital Signs Thresholds)

| المؤشر | بالغين (≥ 18 سنة) | أطفال (1–17 سنة) | رضع (< 1 سنة) |
|--------|-------------------|------------------|---------------|
| **درجة الحرارة** | ≥ 38.0°م = حمى؛ ≥ 40.0°م = حرجة | ≥ 38.0°م = حمى؛ ≥ 39.5°م = حرجة | ≥ 38.0°م = حرجة في < 3 أشهر |
| **نبضات القلب** | < 60 أو > 100 = غير طبيعي؛ > 150 = حرجة | < 60 أو > 120 = غير طبيعي؛ > 160 = حرجة | < 100 أو > 160 = غير طبيعي؛ > 180 = حرجة |
| **ضغط الدم** | < 90/60 = انخفاض؛ > 180/120 = حرجة | < 70/40 = انخفاض؛ > 160/100 = حرجة | < 60/40 = انخفاض شديد |
| **معدل التنفس** | < 12 أو > 20 = غير طبيعي؛ > 30 = حرجة | < 20 أو > 30 = غير طبيعي؛ > 40 = حرجة | < 30 أو > 50 = غير طبيعي؛ > 60 = حرجة |
| **SpO2** | < 95% = غير طبيعي؛ < 92% = عالي الخطورة؛ < 88% = حرجة | < 95% = غير طبيعي؛ < 92% = حرجة | < 95% = غير طبيعي؛ < 90% = حرجة |

---

## قواعد التصنيف العامة (General Triage Rules)

1. **الأولوية دائماً للمؤشرات الحيوية:** إذا كانت المؤشرات الحيوية في نطاق حرج، يتم تصنيف المريض كـ Critical حتى لو كانت الشكوى الرئيسية خفيفة.
2. **الأطفال والرضع:** أي رضيع < 3 أشهر مع حمى ≥ 38.0°م يُصنّف تلقائياً على الأقل High.
3. **الحوامل:** أي ألم بطن أو نزيف أو صداع شديد في حامل > 20 أسبوع يُصنّف على الأقل High.
4. **كبار السن (> 65 سنة):** انخفاض عتبة التصنيف بدرجة واحدة (مثال: ما يُصنّف Medium لبالغ قد يُصنّف High لمسن).
5. **مرضى نقص المناعة:** أي عرض جديد في مريض سرطان/أيدز/متعاطي كورتيزون يُصنّف على الأقل Medium.
6. **الأدوية:** أي رد فعل دوائي جديد (طفح، صعوبة تنفس، تورم) يُصنّف على الأقل High.
7. **التغير المفاجئ:** أي عرض جديد "لم يحدث من قبل" يُصنّف على الأقل Medium.
